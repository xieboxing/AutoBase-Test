## ⚠️ 技术限制与解决方案

### 问题1: Jetpack Compose UI解析失败

**现象**: `uiautomator dump` 返回的XML中元素数量为0或极少

**原因**: Slickorps APP使用Jetpack Compose框架，其渲染机制与原生View不同

**解决方案**:
1. **坐标点击方案**: 使用已知坐标进行点击操作（当前方案）
2. **OCR识别方案**: 截图后使用OCR识别文字位置
3. **图像匹配方案**: 使用图像模板匹配定位元素
4. **Accessibility服务**: 开发辅助功能服务获取元素信息

**推荐方案**: 坐标点击 + OCR识别组合使用

### 问题2: 登录状态保持

**现象**: 测试过程中有时需要重新登录

**解决方案**: 每次测试前检查登录状态，必要时重新登录

## 🧠 智能知识库

系统会自动学习和优化，知识库位于 `data/knowledge/`:

| 文件 | 说明 |
|------|------|
| `elements.json` | 元素定位映射（主定位+备选定位） |
| `failure-patterns.json` | 失败模式库及解决方案 |
