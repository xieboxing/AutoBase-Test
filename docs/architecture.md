## 模块说明

### CLI 层
命令行入口，使用 Commander.js 实现。提供以下命令：
- `web`: 测试网站
- `app`: 测试 APP
- `api`: 测试 API
- `optimize`: AI 优化
- `report`: 报告管理
- `record`: 录制用例

### 核心引擎层
- **Orchestrator**: 测试编排器，总指挥，协调各模块工作
- **TestRunner**: 执行单个测试用例
- **ResultCollector**: 收集测试结果
- **Lifecycle**: 管理测试生命周期
- **Logger**: 日志输出
- **ErrorHandler**: 错误处理
- **EventBus**: 模块间通信

### AI 智能层
- **AiClient**: AI API 客户端，支持 Claude/OpenAI
- **Analyzer**: 页面智能分析
- **CaseGenerator**: 测试用例生成
- **FailureAnalyzer**: 失败原因分析
- **SelfHealer**: 自愈引擎，修复元素定位
- **FlowOptimizer**: 测试流程优化

### 测试执行层
- **Web 测试器**: 基于 Playwright
  - PcTester: PC Web 测试
  - H5Tester: H5 移动端测试
  - FormTester: 表单测试
  - NavigationTester: 导航测试
  - 等等...

- **APP 测试器**: 基于 Appium
  - InstallTester: 安装测试
  - LaunchTester: 启动测试
  - UiTester: UI 交互测试
  - GestureTester: 手势测试
  - 等等...

- **其他测试器**
  - ApiTester: API 测试
  - SecurityTester: 安全测试
  - A11yTester: 无障碍测试
  - PerformanceTester: 性能测试

### 爬虫/探索层
- **WebCrawler**: 爬取网站所有页面
- **AppCrawler**: 遍历 APP 所有页面
- **PageSnapshot**: 生成页面快照

### 知识库层
- **TestHistory**: 测试历史记录
- **FailurePatterns**: 失败模式库
- **ElementMapping**: 元素定位映射（自愈用）

### 报告生成层
- **HtmlReporter**: HTML 可视化报告
- **JsonReporter**: JSON 数据报告
- **MdReporter**: Markdown 报告

## 数据流

```
1. 用户提供 URL/APK
       │
       v
2. Orchestrator 初始化环境
       │
       v
3. Crawler 探索页面/页面
       │
       v
4. AI Analyzer 分析页面
       │
       v
5. CaseGenerator 生成测试用例
       │
       v
6. TestRunner 执行测试用例
       │
       v
7. ResultCollector 收集结果
       │
       v
8. ReportGenerator 生成报告
       │
       v
9. KnowledgeBase 保存历史
```

## 自愈机制流程

```
元素定位失败
       │
       v
截图当前页面
       │
       v
查询知识库历史定位
       │
       v
发送截图+DOM给AI
       │
       v
AI返回新选择器
       │
       v
用新选择器重试
       │
       v
成功？──是──> 更新知识库
       │
       否
       │
       v
标记需人工介入
```

## 扩展机制

### 添加新的测试器

1. 在 `src/testers/` 下创建新目录
2. 实现测试器类，继承基础接口
3. 在 `index.ts` 中导出
4. 在 Orchestrator 中注册

示例：

```typescript
// src/testers/custom/custom-tester.ts
export class CustomTester {
  async initialize() { /* ... */ }
  async runTest(testCase: TestCase) { /* ... */ }
  async close() { /* ... */ }
}
```

### 添加新的 AI Prompt

1. 在 `src/ai/prompts/` 创建新的 prompt 文件
2. 实现构建函数和解析函数
3. 在 AI 模块中调用

示例：

```typescript
// src/ai/prompts/custom.prompt.ts
export function buildCustomPrompt(params: CustomParams): string {
  return `分析以下内容...`;
}

export function parseCustomResult(response: string): CustomResult {
  return JSON.parse(response);
}
```
